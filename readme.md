Este projeto desenvolve um modelo de regressao linear para estimar o numero de cliques em campanhas de anuncios com base na quantidade de anuncios exibidos.

Objetivo
Prever o desempenho de campanhas futuras utilizando dados historicos de marketing digital.

Etapas do Projeto
Carregamento do dataset e exibicao de informacoes gerais.

Analise exploratoria com grafico de dispersao.

Separacao de variaveis e divisao entre treino e teste.

Treinamento do modelo de Regressao Linear.

Visualizacao da reta de regressao e metricas de erro.

Requisitos
Python 3 e bibliotecas:

pandas

matplotlib

scikit-learn

Como usar
Mantenha o arquivo marketing.csv na mesma pasta do script.

Execute o arquivo python para visualizar os graficos e os resultados do modelo.

Resultados
O modelo gera uma linha de tendencia que permite calcular a estimativa de cliques para diferentes volumes de anuncios.
